Sell-in / COP 목표 수동 입력

해당 월의 Sell-in 실적(USD/KRW)과 COP(목표) 금액(KRW)을 직접 입력합니다. 하나만 입력해도 됩니다. USD/KRW 중 하나를 입력하면 환율 1,480원 기준으로 나머지가 자동으로 채워집니다 (필요하면 직접 수정 가능).
이 페이지는 data.json 파일의 내용을 보여줍니다. 데이터 갱신은 담당자가 data.json 파일을 교체(재배포)해야 반영됩니다.
